# Portfólio visual (HTML estático)

Esta pasta apresenta os projetos do repositório em uma **página única** (`index.html`) com o mesmo modelo visual:

- **Foto de perfil** no topo (`assets/perfil.png`)
- **Seta para baixo** convidando a rolar até os projetos (`assets/icone-seta-baixo.png`)
- **Ícone WhatsApp** no projeto de chatbot e no rodapé de contato (`assets/icone-whatsapp.png`)

## Como visualizar

Abra o arquivo no navegador (duplo clique ou arrastar `index.html` para o Chrome/Edge/Firefox).

Ou, na pasta `automacoes`:

```powershell
start portfolios\index.html
```

## GitHub Pages (opcional)

No repositório GitHub: **Settings → Pages →** escolha branch `main` e pasta `/portfolios` ou `/root` conforme configurar. Se publicar só a pasta `portfolios`, ajuste os caminhos ou coloque `index.html` na raiz do site.

## Ajustes recomendados

1. No rodapé de `index.html`, substitua `https://wa.me/5511999999999` pelo seu número (DDI + DDD + número, sem `+` ou espaços).
2. Atualize textos em primeira pessoa se quiser personalizar a bio.
