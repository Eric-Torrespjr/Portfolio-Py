# Automações Python — portfólio (APIs & integração)

Três exemplos focados em **automação**, **APIs REST** e **integração de dados**, alinhados a perfis do tipo *Desenvolvedor Python (Automação & APIs)*.

| Pasta | Descrição resumida | Documentação |
|--------|-------------------|--------------|
| [`integrador_dados_api/`](integrador_dados_api/) | Consome API pública, normaliza dados, exporta CSV e gera log. | [README](integrador_dados_api/README.md) |
| [`servico_rest/`](servico_rest/) | API FastAPI com CRUD, health check e documentação `/docs`. | [README](servico_rest/README.md) |
| [`painel_gui/`](painel_gui/) | Interface tkinter que dispara o integrador e testa a API local. | [README](painel_gui/README.md) |
